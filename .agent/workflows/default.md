# Default Workflow Configuration

This workflow enables automatic command execution (turbo mode) for all steps in this project.

## Common Operations

1. Run git status
```
git status
```

2. Stage all changes
```
git add -A
```

3. Commit changes
```
git commit -m "message"
```

4. Push to remote
```
git push
```

5. Pull from remote
```
git pull
```

6. Run Python scripts
```
python script.py
```

7. Run tests
```
pytest
```

8. Install dependencies
```
pip install -r requirements.txt
```
